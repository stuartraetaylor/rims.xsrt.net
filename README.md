# Wheel Tools

Wheel offset and fitment comparison calculators.

**https://rims.xsrt.net/**

## Development

```bash
npm install
npm run dev        # Dev server with live reload at http://localhost:8080
npm test           # Run Jest unit tests
npm run build      # Production build to _site/
```

## Docker

```bash
docker compose up -d --build   # Serve production build at http://localhost:8080
```
